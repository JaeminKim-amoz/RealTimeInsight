# Production Frontend Source

This folder will replace the CDN/Babel prototype in `project/app` with a production React/Tauri frontend.

The prototype remains the visual and interaction reference. New production components should be written test-first and should preserve the high-density workstation behavior:

- channel explorer and mapping editor
- docked multi-panel workspace
- linked cursor across charts/map/video/graph
- receiver diagnostics and quality policies
- MATLAB MCP, SimDIS, Ollama, export, and graph root-cause workflows
